Leaf Disease Classification

Plant Disease Classification Using Deep Learning

This project was made by Parth Dedhia, undergrad student at VIT-C 2025 batch, as a project for Machine Vision.

Introduction

This project uses a Convolutional Neural Network to classify the disease in a leaf image. The model was trained on the PlantVillage dataset, which contains images of diseased and healthy leaves of various plants. The model was trained on a GPU for 15 epochs and achieved an accuracy of 97.5% on the test set.
